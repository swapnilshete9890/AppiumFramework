# Appium Automation Framework

This repository contains an Android automation framework built using **Appium**, **Java**, **TestNG**, and **Page Object Model (POM)**.  
It includes reusable page classes, TestNG listeners, and Extent Reports for test execution results.

## Tech Stack
- Java
- Appium
- TestNG
- Maven
- Page Object Model (POM)
- Extent Reports
